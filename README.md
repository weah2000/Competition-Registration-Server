# register-server

#### 介绍
一个简单报名登记+查询系统
基于nodejs+mongodb

#### 软件架构
1.  nodejs
2.  mongodb
3.  bootstrap

#### 安装教程

1.  npm install
2.  修改 config文件下default.conf文件,设置mongodb的配置
3.  npm start

#### 使用说明

1.  浏览器打开http://localhost:5000
2.  注册: http://localhost:5000
3.  查询: http://localhost:5000/query.html
